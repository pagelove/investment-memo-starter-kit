Pagelove - what it's all about

Pagelove is a new way to think about building web applications, that's more like the old way.

Because we have come to think of the web as an application delivery mechanism, the amount of tooling, frameworks, and libraries that are needed to just get started with anything is truly staggering. With the rise of node.js APIs and single page applications, we have moved away from the foundational ideas of the web, to a world where the web is just a big dumb pipe.

Pagelove enables surgical DOM manipulations using standard HTTP and HTML. This makes for much simpler applications, that don't require complex state management in the client.

But radically simpler applications aren't a business. The business of pagelove comes when you realize that because pagelove is just HTML and HTTP, it lives at the edge.

Pagelove is the perfect platform for edge computing, because the only state is the document state.

The power of pagelove is to surgically alter pages. Should work....
